## 当前保留内容

### 在线源码查找

需要快速跳转到某个 C/C++ 标准库或开源项目的源码定义时，可以先用：

- <https://codebrowser.dev/>

它适合「我只想看符号定义在哪、不想本地拉一份完整代码」的场景。

### git 基础命令片段

原始笔记里只留了几条最基础的步骤，这里补上对应说明：

```bash
# 1. 初始化仓库（在当前目录下）
git init

# 2. 添加所有改动到暂存区
git add .
```

`git add .` 会把当前目录下所有变化（新增、修改、删除）一起加入暂存区，提交前最好用 `git status` 再确认一遍。
