# tools/

杂项资源与脚本片段。这些文件历史上被误放在 `_posts/` 里，但它们既不是 Markdown 文章，也不会被 Jekyll 渲染。统一搬到这里集中存放。

## 目录内容

| 文件 | 用途 |
|------|------|
| `analyze_code.py` | Python 脚本片段 |
| `cpp_syllabus.pdf` | C++ 学习大纲 PDF |
| `curl-brpc-debug` | curl 调试 brpc 服务的笔记片段 |
| `dos2unix.py` | dos2unix 转换脚本 |
| `gtest-cheetsheet` | gtest 命令速查 |
| `hast_test.python` | Python 脚本片段 |
| `myhttp.py` | HTTP 工具脚本 |
| `python-8进制转字符串` | Python 八进制转字符串片段 |
| `redis-client` | redis-client 笔记片段 |
| `singleton.h` | 单例模式头文件示例 |
| `windows-内存泄漏排查.mddd` | Windows 内存泄漏排查笔记（扩展名异常） |
| `wsl-bridged.zip` | WSL 桥接配置压缩包 |
| `未命名绘图.drawio` | drawio 草图源文件 |

> 这些文件不参与站点构建。如果需要将其中的笔记片段转为正式文章，请新建带 `YYYY-MM-DD-` 前缀的 `.md` 文件放入 `_posts/` 对应分类目录。
